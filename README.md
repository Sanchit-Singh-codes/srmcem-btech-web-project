# 🏫 SRMCEM B.Tech 3rd Year Web Project

> **Academic web portal project built as part of the B.Tech 3rd Year curriculum at SRMCEM.**

Shri Ramswaroop Memorial College of Engineering and Management (SRMCEM) academic portal project. Contains clean, responsive vanilla HTML5 pages demonstrating user management flows, contact routing, information structure, and college portals.

---

## ✨ Features

- 🏠 **Main Homepage (`index.html`)**: Landing page introducing the portal, showcasing the campus structure and linking internal directories.
- ℹ️ **About Page (`about.html`)**: Details college details, courses, and engineering infrastructure.
- 📞 **Contact Routing (`contact.html`)**: Forms and detail cards for student support and query submissions.
- 👥 **Employee Directory (`employee.html`)**: Simple data visualization tables for college administration and faculty staff roster.
- 🌐 **Responsive Design**: Designed with standard viewport metadata for scaling across mobile devices.

---

## 🛠️ Tech Stack

- **Markup**: HTML5 (Semantic tagging)
- **Assets**: Local college campus graphics (`srmcem.jpg`)

---

## 🚀 How to Run

Since this is a lightweight static web project, it runs directly on any browser.

1. Clone the repository:
   ```bash
   git clone https://github.com/Sanchit-Singh-codes/srmcem-btech-web-project.git
   cd srmcem-btech-web-project
   ```

2. Run:
   - Double click `index.html` to load it in your browser.
   - Or, open it with VS Code Live Server extension.
